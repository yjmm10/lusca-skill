## [1.3.0] — 2026-07-14

### Changed
- 设计理念修正：六维精读与五视角批判定位为**解读时的内化指导**，而非笔记的固定输出结构
- 模板「批判性评估」节改为**自然叙述**：去掉 5.1–5.4 框架化子节与视角标签，只保留综合可信度收尾；反方论证改为可选引出（非强制）
- SKILL.md：主流程 Step 3/4 标注「指导」、六维与五视角摘要节加定位说明、阅读笔记结构与模板使用约定同步
- references/reading-dimensions.md、critique-lenses.md 开头明确「是指导框架、不是输出结构」

## [1.2.0] — 2026-07-14

### Changed
- 模板重构：元信息只在 YAML frontmatter，正文不再单列元信息节
- 批判性评估**有机融合**六维精读与五视角批判：按"核心 claim 是否站得住 → 方法与结果是否可信 → 贡献与论证质量 → 综合可信度"主线编织，不再列两张独立的表；反方论证单独成块
- 额外节精简：复现性 / Limitations / 引用建议 / 盲点 / action 合并为「Limitations 与复现性」「可复用与后续」两节，每条一行
- 论文内容解读（研究问题 / 贡献 / 方法 / 结果）保持饱满，不简化
- 同步更新「阅读笔记结构」「模板使用约定」节

## [1.1.0] — 2026-07-14

### Added
- 阅读笔记模板大幅增强：YAML frontmatter（便于检索）、TL;DR、复现性 checklist、引用建议、盲点与待澄清、后续 action、综合可信度判断
- 新增「模板使用约定」小节：逐节填写不删节、frontmatter 必填、事实 vs 评估区分、严重度必标、不确定必标

### Changed
- arXiv ID 抓取策略：优先 `arxiv.org/html/<id>` 全文 HTML，无则下载 PDF 用 Read 分块读，abs 仅补元信息
- 主流程 Step 5：明确「严格对照模板逐节填写，空节标 n/a 不删节」
- 阅读笔记结构节：精简为指向模板，避免与模板重复
- 质量底线：补「不替代检索」边界——需查 related work 时引导到 `lusca-paper-search`
